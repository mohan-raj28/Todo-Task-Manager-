# Todo Task Manager Node.js + MySQL Backend

## Setup

1. Install dependencies:
   ```sh
   npm install
   ```
2. Create a MySQL database (default: `todo_db`).
3. Copy `.env` and set your MySQL credentials if needed.
4. Start the server:
   ```sh
   node index.js
   ```
